
# Mini SOC with Wazuh
This project simulates a small Security Operations Center (SOC) using Wazuh.

## Objectives
- Deploy a Wazuh SIEM
- Collect logs from Linux and Windows endpoints
- Detect brute force and authentication attacks
- Apply basic response actions

## Tools
- Wazuh
- Ubuntu Server
- Windows Server / Windows 10
- VirtualBox

## Lab Architecture
- Wazuh Manager
- 1 Linux endpoint
- 1 Windows endpoint
